# 📱QR Code Generator

"A modern QR Code Generator that instantly converts text and URLs into downloadable scannable QR codes using HTML, CSS, and JavaScript."

---

## 🚀 Features

* Generate QR codes instantly
* Supports both text and URLs
* Download QR codes with a single click
* Responsive and clean UI
* Smooth animations and interactions
* User-friendly design

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* QR Code API

---

## 📚 Concepts Implemented

* DOM Manipulation
* Event Handling
* API Integration
* classList
* createElement('a')
* click()
* Dynamic Image Generation

---

## 📷 How It Works

1. Enter any text or URL
2. Click on **Generate QR**
3. QR code is generated instantly
4. Download the QR code using the download button
5. Scan the QR code to access the content
